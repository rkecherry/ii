A subnet mask is a number that distinguishes the network address and the host address within an IP address. A subnet is a smaller network within a network that requires a subnet mask. Subnetting is the process of dividing a network into two or more subnets. Its primary function is to make the routing of data within a network more efficient and secure. Subnetting also helps make better use of IPv4 addresses. When you connect a device to a network, the network assigns an IP address to the device. That IP address consists of two parts: the network portion and the host portion. The network portion of the IP address identifies the overall network while the host portion identifies the device.
